Welcome to the Community-Powered AI Threat Detector documentation.

Overview

The AI threat Detector is a tool designed to help users identify and flag potentially misleading or harmful content on the web. It consists of a browser extension and a web dashboard.

How it Works

Users can install the browser extension to:

  • View flags on visited pages.
  • Submit new flags for suspicious content.

The dashboard you are currently viewing aggregates all flagged content for analysis.

Data Sources

This dashboard pulls data directly from our Supabase database. It displays:

  • Flagged Links: URLs that have been reported as suspicious.
  • Flagged Content: Specific text or media elements within a page that have been flagged.
